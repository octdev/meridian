# Weekly Outtake

Rolling 7-day view of completed tasks. For permanent weekly records, see `Process/Weekly/`.

## Action Items — Urgent
```tasks
done
done after 7 days ago
description includes !!
path includes /Daily/
group by done
sort by done
```

## Action Items — Standard
```tasks
done
done after 7 days ago
description includes !
description does not include !!
path includes /Daily/
group by done
sort by done
```

## Open Loops Closed
```tasks
done
done after 7 days ago
description includes ~
path includes /Daily/
group by done
sort by done
```

## Review Items Processed
```tasks
done
done after 7 days ago
description includes >>
path includes /Daily/
group by done
sort by done
```
